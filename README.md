# YouTube2-analogue

Это аналог YouTube2, написано на Python 

Как использовать мой проект:

Вам нужно 
- Установить Python
- Установить необходимые библиотеки (
```
pip install flask
```
```
pip install flask_sqlalchemy
```
```
pip install flask_login
```
)

Запуск в терменале 
```
python3 app.py
```
